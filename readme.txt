Create content blocks that can be attached to sidebars on specific pages

== Description ==

This plugin emulates a standard Drupal feature called Blocks.

It creates a block content type which supports WYSIWYG editing that can
be attached to any theme-defined sidebars on any pages or set of pages.

Blocks supports multi-site or single-site installations with no configuration.

Source also available on github at: https://github.com/jtlabak-uw/wp-blocks

== Installation ==

1. Unzip "Blocks" archive and put all files into a folder like "/wp-content/plugins/blocks/"
1. Activate the plugin through the 'Plugins' menu in WordPress
